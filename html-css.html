Portfólio HTML5 & CSS3

Exemplos de projetos usando HTML5 e CSS3

Projeto: Página simples responsiva

Uma página que adapta o layout para dispositivos móveis usando media queries.


/* Exemplo CSS */
@media (max-width: 600px) {
  body {
    background-color: #f0f0f0;
  }
}

Projeto: Formulário estilizado

Formulário com inputs customizados e efeitos visuais com CSS.


input[type="text"] {
  border: 1px solid #ccc;
  padding: 8px;
  border-radius: 4px;
}
input[type="text"]:focus {
  border-color: #2b7de9;
  outline: none;
}
