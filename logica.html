Portfólio Lógica com C e Python (CS50)

Exemplos de código para demonstrar lógica de programação

Exemplo em C


#include <stdio.h>

int main() {
  int num;
  printf("Digite um número: ");
  scanf("%d", &num);
  if(num % 2 == 0) {
    printf("Número par\n");
  } else {
    printf("Número ímpar\n");
  }
  return 0;
}

Exemplo em Python


num = int(input("Digite um número: "))
if num % 2 == 0:
    print("Número par")
else:
    print("Número ímpar")
