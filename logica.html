Portfólio Lógica com C e Python (CS50)

Exemplos de código e mini projeto interativo com a mesma lógica em JS

Exemplo em C


#include <stdio.h>

int main() {
  int num;
  printf("Digite um número: ");
  scanf("%d", &num);
  if(num % 2 == 0) {
    printf("Número par\n");
  } else {
    printf("Número ímpar\n");
  }
  return 0;
}

Exemplo em Python


num = int(input("Digite um número: "))
if num % 2 == 0:
    print("Número par")
else:
    print("Número ímpar")

Projeto interativo: Verificar par/ímpar e calcular fatorial

Explicação: A lógica para verificar se um número é par ou ímpar e para calcular o fatorial é a mesma em C, Python e JavaScript. Nesta página, mostramos os códigos em C e Python e disponibilizamos um mini projeto interativo em JavaScript que aplica a mesma lógica, para você testar diretamente aqui no navegador sem precisar instalar nada. Isso ajuda a entender que a lógica de programação é universal, independentemente da linguagem usada.

Tecnologias usadas

  • C: Linguagem compilada, usada para programas que exigem alta performance e controle de baixo nível.
  • Python: Linguagem interpretada, muito usada para ensino de lógica e desenvolvimento rápido.
  • JavaScript: Linguagem de programação que roda no navegador para criar interatividade.
  • HTML5 e CSS3: Linguagens para estruturar e estilizar a página web.
